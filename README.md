# Moodle - snippets & autocomplete

This extension for Visual Studio Code adds Moodle snippets and autocompletion of functions, classes.

## Features

All methods for GLOBAL variables:

- `$OUTPUT`

Also there are file comments

## Requirements

VSCode 1.30.0
